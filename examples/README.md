# Sample Images Placeholder

This directory contains example images for testing the application.

## Add Your Own Images

Place your test images here with names like:
- `cat.jpg`
- `dog.jpg`
- `car.jpg`
- `bird.jpg`
- `flower.jpg`

## Image Requirements

- **Format**: JPEG or PNG
- **Size**: At least 224x224 pixels
- **Content**: Clear, well-lit images work best

## Download Sample Images

You can download sample images from:
- [ImageNet Sample Images](http://image-net.org/)
- [Unsplash](https://unsplash.com/) (free high-quality images)
- [Pexels](https://www.pexels.com/) (free stock photos)

## Tips for Best Results

1. Use images with clear subjects
2. Avoid heavily filtered or edited images
3. Ensure good lighting and contrast
4. Center the main subject in the frame
