# Example Images

This directory contains example images for testing the application.

## Current Examples

- `cat.jpg` — Cat image
- `dog.jpeg` — Dog image
- `bird.jpg` — Bird image
- `car.jpg` — Car image
- `flower.jpg` — Flower image

## Add Your Own Images

Place your test images here. Supported formats:
- **JPEG** (`.jpg`, `.jpeg`)
- **PNG** (`.png`)
- **WEBP** (`.webp`)
- **BMP** (`.bmp`)
- **TIFF** (`.tiff`)

## Image Requirements

- **Size**: At least 224×224 pixels recommended
- **Content**: Clear, well-lit images work best
- **Subject**: Center the subject in the frame

## Download Real Sample Images

For better predictions, replace these with real photos from:
- [Unsplash](https://unsplash.com/) (free high-quality images)
- [Pexels](https://www.pexels.com/) (free stock photos)

## Tips for Best Results

1. Use images with clear subjects
2. Avoid heavily filtered or edited images
3. Try different models — EfficientNetB0 & ResNet50 are usually most accurate
3. Ensure good lighting and contrast
4. Center the main subject in the frame
